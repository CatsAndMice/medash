```ts
medash.trim(str:string):string
```

修剪字符串两端的空字符
  
**添加版本**  
v3.3.6

**参数**   
str(any):任意类型值

**返回**  
(string): 已两端已修剪的字符串

**例子**  
```js
trim(' trim ')//trim
```
